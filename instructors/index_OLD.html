Instructor Support

Learning Web Design has been a leading textbook for entry-level web design and production courses since its first edition in 2001. The “classroom-in-a-book” structure of the text, complete with quizzes and exercises, provides a ready-made course foundation. For the first time, an Instructor Guide and slide decks for all chapters will be available for web design teachers at universities, high schools, boot camps, and wherever you may be using Learning Web Design as your textbook.

The guide and slides are in production now and should be ready in September. If you would like to be contacted when they are available, please email the author at .

COMING SOON!! Instructor’s Guide
This booklet introduces the structure and strategy for using Learning Web Design in the classroom. It contains a general overview, equipment and software requirements, and how to use the book as a reference. For every chapter there is a summary, learning goals, key terms, and a list of exercises from the book.
COMING SOON!! Slide decks
Every chapter has a slide deck that provides an at-a-glance view of the content for use during class discussions. Slide decks (available in Keynote, Powerpoint, and PDF formats) can be edited to customize them for your particular class structure.
